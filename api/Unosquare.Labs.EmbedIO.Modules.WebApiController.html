    SetDefaultHeaders()

    Sets the default headers to the Web API response. By default will set:

    Expires - Mon, 26 Jul 1997 05:00:00 GMT LastModified - (Current Date) CacheControl - no-store, no-cache, must-revalidate Pragma - no-cache

    Previous values are defined to avoid caching from client.
