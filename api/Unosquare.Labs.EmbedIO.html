    Represents our tiny web server used to handle requests.

    This is the default implementation of IWebServer and it's ready to select the IHttpListener implementation via the proper constructor.

    By default, the WebServer will use the Regex RoutingStrategy for all registered modules (IWebModule) and EmbedIO Listener (HttpListenerMode).
